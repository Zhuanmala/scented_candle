# Blind Scent Test Questionnaire

Do not explain the brand story before the first round. Let the participant smell the samples first.

## Participant Metadata

- Participant code:
- City:
- Age range:
- Gender optional:
- Relationship to project: close friend / acquaintance / stranger / design-art circle / other
- Usually buys scented candles: never / sometimes / often

## Round 1: Scent Only

For each sample A/B/C:

1. First impression in 3 words.
2. How premium does it smell? 1-5.
3. Does it smell cheap or synthetic? 1-5.
4. Is it too strong? yes / no.
5. Is it too weak? yes / no.
6. Where would you use it?
7. Would you buy it for yourself? yes / maybe / no.
8. Would you give it as a gift? yes / maybe / no.

## Round 2: Ranking

1. Which sample do you like most?
2. Why?
3. Which sample do you like least?
4. Why?
5. Which sample is best for autumn/winter?
6. Which sample is best as a gift?
7. Which sample is most memorable?

## Round 3: Packaging and Price

After showing packaging mockup:

1. What price does this look like?
2. Would you pay EUR 29?
3. Would you pay EUR 34?
4. Would you pay EUR 39?
5. What would stop you from buying?
6. What would make you trust the brand more?

## Round 4: Action Signal

1. Would you like to be notified if this launches?
2. Would you leave your email?
3. Would you pay a refundable EUR 5-10 deposit to reserve one?
4. Would you introduce this to someone who might buy it?

## Valid Demand Signal

Count as strong signal:
- Email left.
- Deposit willingness.
- Asks when available.
- Names a specific gift recipient.
- Offers referral.

Do not count as strong signal:
- "Interesting."
- "Looks nice."
- "Good idea."
- Social media likes without signup.
